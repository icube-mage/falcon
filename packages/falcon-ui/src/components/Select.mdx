`Select.js` and `Option` components render as native `<select>` and `<option>` elements with basic styling applied,
accept all of the native element props (`onChange`, `value` etc) as well as [responsive, theme-based style props](/responsive-props)
for styling purposes.

```javascript
import { Select.js, Option } from '@deity/falcon-ui';
```
