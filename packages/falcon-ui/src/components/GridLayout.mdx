`GridLayout` component is rendered by default as a `<div>` with `display: grid` ([CSS Grid Layout](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_Grid_Layout)) and accepts other [responsive, theme-based style props](/responsive-props) such as (`gridTemplateColumns`, `gridTemplateColumns` etc).

```javascript
import { GridLayout } from '@deity/falcon-ui';
```
