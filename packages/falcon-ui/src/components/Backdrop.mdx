`Backdrop` component is useful for handling "click outsite to close" functionality for modals, [sidebars](/components-sidebar) etc.

When visible (`visible={true}`) it renders as full viewport element with a semi-transparent background.

```javascript
import { Backdrop } from '@deity/falcon-ui';
```

<Playground>
  <Toggle initial={false}>
    {({ on, toggle }) => (
      <React.Fragment>
        <Backdrop visible={on} onClick={toggle} />
        <Button onClick={toggle}>Show Backdrop</Button>
      </React.Fragment>
    )}
  </Toggle>
</Playground>

`Backdrop` can also be rendered as a [`portal`](/components-portal) to avoid issues with elements nesting.
<Playground>
  <Toggle initial={false}>
    {({ on, toggle }) => (
      <React.Fragment>
        <Backdrop as={Portal} visible={on} onClick={toggle} />
        <Button onClick={toggle}>Show Backdrop</Button>
      </React.Fragment>
    )}
  </Toggle>
</Playground>
