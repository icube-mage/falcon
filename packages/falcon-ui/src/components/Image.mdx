`Image` component is rendered as an `<img>` and accepts all of the `<img>` props as well as all of the [responsive, theme-based style props](/responsive-props).
By default it applies `object-fit: contain` and `max-width: 100%` in order to fit parent container and still maintain image's aspect ratio.
        

```javascript
import { Image } from '@deity/falcon-ui';
```
