`Dropdown` component along with `DropdownLabel`, `DropdownMenu` and `DropdownMenuItem` can be used for creating toggleable dropdowns in declarative way.

```javascript
import { Dropdown, DropdownLabel, DropdownMenu, DropdownMenuItem } from '@deity/falcon-ui';
```

<Playground>
  <Box css={{height: 150}}>
    <Value initial="acending">
       {({ value, set }) => (
          <Dropdown onChange={set}>
            <DropdownLabel>
                Sort by: {value}
            </DropdownLabel>
            <DropdownMenu>
              <DropdownMenuItem value="ascending">
                  Ascending
              </DropdownMenuItem>
              <DropdownMenuItem value="descending">
                  Descending
              </DropdownMenuItem>
            </DropdownMenu>
          </Dropdown>
        )}
    </Value>
  </Box>
</Playground>

`Dropdown`,`DropdownLabel`, `DropdownMenu` and `DropdownMenuItem` components all accept all of the [responsive, theme-based style props](/responsive-props).
