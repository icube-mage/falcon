`Icon`'s component main purpose is to render icons defined in `ThemeProvider`'s `icons` section.
It's being used internally in other `falcon-ui` components such as `Checkbox` to render default icons 
with option to overrride them using `ThemeProvider`.

```javascript
import { Icon } from '@deity/falcon-ui';
```
