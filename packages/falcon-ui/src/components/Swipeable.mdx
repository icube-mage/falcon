`Swipeable` and `SwipeableItem` components are designed as foundation for building image galleries, carousels
in very straightforward way using [`CSS Scroll Snap` capabilities](https://developers.google.com/web/updates/2018/07/css-scroll-snap).
Both accept [responsive, theme-based style props](/responsive-props) for styles customization.

```javascript
import { Swipeable, SwipeableItem } from '@deity/falcon-ui';
```
