Both `Breadcrumb` and `Breadcrumbs` components accept all of the [responsive, theme-based style props](/responsive-props) that allow further styling customizations.

 ```javascript
import { Breadcrumbs, Breadcrumb } from '@deity/falcon-ui';
```

`Breadcrumb`'s default separator is implemented as CSS [::after](https://developer.mozilla.org/en-US/docs/Web/CSS/::after) pseudo element and
 can be easily changed via custom theme provided to `ThemeProvider`.

<Playground>
  <ThemeProvider theme={themeWithCustomBreadcrumbSeparator}>
    <Breadcrumbs>
      <Breadcrumb>
        <Link>One</Link>
      </Breadcrumb>
      <Breadcrumb>
        <Link>Two</Link>
      </Breadcrumb>
      <Breadcrumb>
        <Link>Three</Link>
      </Breadcrumb>
    </Breadcrumbs>
  </ThemeProvider>
</Playground>

```javascript
const themeWithCustomBreadcrumbSeparator = createTheme({
  components: {
    breadcrumb: {
      css: {
        '::after': {
          content: '"/"'
        }
      }
    }
  }
});
```
